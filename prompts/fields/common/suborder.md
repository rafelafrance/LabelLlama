It is distinct from order (a higher rank) and family (a lower rank). Suborder may appear with an explicit label such as 'subord.', 'subord. Cicindelina', or simply as a standalone taxonomic name. Return the suborder name only — do not include the order, family, genus, specific epithet, authorship citations, or any other taxonomic rank. If no suborder is stated, return an empty string.
