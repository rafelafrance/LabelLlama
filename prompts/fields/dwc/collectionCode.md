`collectionCode` (str): Extract the collection code. The name, acronym, coden, or initialism identifying the collection or data set from which the record was derived. This is often given within a colon separated triplet with the institution code, collection code, and catalog number like: "ALMNH:Ento:137097" or "ALMNH:Ento:137005"; in these cases the institution code is the second part of the triplet "Ento". Standalone examples: "Mammals", "Hildebrandt", "EBIRD", "VP". If no collection code is present, return an empty string.
