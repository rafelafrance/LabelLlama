`institutionCode` (str): Extract the institution code. The name (or acronym) in use by the institution having ownership of the object(s) or information referred to in the record. This is often given within a colon separated triplet with the institution code, collection code, and catalog number like: "ALMNH:Ento:137097" or "ALMNH:Ento:137005"; in these cases the institution code is the first part of the triplet "ALMNH". Standalone examples: "NPS", "APN", "InBio", "ALMNH". If no institution code is present, return an empty string.
